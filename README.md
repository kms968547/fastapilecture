# Backend API 과제 프로젝트

이 프로젝트는 FastAPI를 활용하여 구현된 백엔드 API 서버입니다. 과제 요구사항에 맞춰 핵심적인 API 기능들을 보여주는 데 중점을 두었으며, 별도의 데이터베이스 연결 없이 서버 메모리 안에서 데이터를 관리합니다.

## 프로젝트의 특징

*   **다양한 HTTP 메소드 API**: POST, GET, PUT, DELETE 각각 2개씩, 총 8개의 API 엔드포인트가 구현되어 있습니다. 이를 통해 기본적인 CRUD(생성, 조회, 수정, 삭제) 작업을 수행하는 방법을 보여줍니다.
*   **요청 처리 미들웨어**: 모든 API 요청이 들어올 때마다 요청 정보를 기록하고, 실제 처리 시간을 측정하여 응답에 포함하는 미들웨어가 적용되어 있습니다.
*   **일관된 응답 형식 및 상태 코드**: 모든 API 응답은 정해진 JSON 형식으로 통일되어 있으며, 2xx(성공), 4xx(클라이언트 오류), 5xx(서버 오류) 등 다양한 HTTP 상태 코드를 상황에 맞게 사용합니다.

## 프로젝트 실행 방법

프로젝트를 실행하려면 다음 단계를 따라주세요.

1.  **가상 환경 만들기 및 활성화**:
    프로젝트 폴더(이 `README.md` 파일이 있는 곳)에서 터미널을 열고 다음 명령어를 입력합니다.

    ```bash
    # 가상 환경이 아직 없다면 새로 만듭니다.
    python -m venv venv

    # 가상 환경을 활성화합니다. (Windows 환경 기준)
    .\venv\Scripts\activate
    ```

2.  **필요한 라이브러리 설치**:
    가상 환경이 활성화된 상태에서 다음 명령어를 입력하여 프로젝트에 필요한 모든 라이브러리들을 설치합니다.

    ```bash
    pip install -r requirements.txt
    ```

3.  **API 서버 실행**:
    이제 다음 명령어로 FastAPI 서버를 시작할 수 있습니다.

    ```bash
    uvicorn app_fastapi:app --reload
    ```

    서버가 성공적으로 시작되면, 웹 브라우저에서 `http://127.0.0.1:8000` 주소로 접속해 보세요.
    자동으로 생성되는 API 문서 페이지 `http://127.0.0.1:8000/docs` 에서는 각 API의 상세한 기능과 사용법을 확인하고 직접 테스트해 볼 수 있습니다.

## 중요 참고 사항

이 프로젝트는 데이터를 서버 메모리에서만 관리하기 때문에, 서버를 껐다 켜면 저장된 모든 데이터는 초기화됩니다. 이 점을 참고하여 테스트해 주세요.
