애플리케이션 실행에 지속적으로 문제가 발생하여 죄송합니다. 현재 환경에서 명령줄 인수를 구문 분석하는 데 어려움이 있는 것으로 보입니다.

가장 확실한 방법은 환경 변수를 먼저 설정한 다음, 애플리케이션 실행 명령을 별도로 실행하는 것입니다.

**Windows Command Prompt에서 다음 두 명령을 순서대로 실행해 주십시오.**

1.  **환경 변수 설정 (SSH 터널이 활성화되어 있어야 함):**
    ```cmd
    set SPRING_PROFILES_ACTIVE=test
    set DB_USERNAME=guest
    set DB_PASSWORD=your_mysql_password_here
    ```
    *   `your_mysql_password_here` 부분은 Jcloud MySQL 서버의 'guest' 사용자에 대한 실제 비밀번호로 교체해야 합니다.

2.  **애플리케이션 실행:**
    ```cmd
    ./mvnw spring-boot:run
    ```

**다시 한번 강조하지만, 이 명령들을 실행하기 전에 SSH 터널이 `localhost:3307`로 올바르게 포워딩되어 활성화되어 있는지 확인해 주십시오.**

이 방법으로 애플리케이션이 성공적으로 시작될 것으로 예상합니다. 결과를 알려주십시오.
